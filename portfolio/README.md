# 🌐 IoT Summer School 2026 – Portfolio

## 📖 Overview

This portfolio is a collection of everything I worked on and learned during the **IoT Summer School 2026**. Over the course of the program, I got hands-on experience with embedded systems, Arduino and ESP32 programming, and building real-world IoT applications.

Instead of just theory, the focus was on actually **building things** — from simple LED circuits to complete smart systems using sensors, communication protocols, and cloud integration. This portfolio reflects that journey and the practical skills I developed along the way.

---

## 🛠️ Skills I Developed

### 🔌 Embedded Systems
- Programming with Arduino and ESP32  
- Working with digital and analog inputs/outputs  
- Using PWM for control applications  
- Understanding interrupts and timers  

---

### 🌡️ Working with Sensors
- Temperature & humidity sensing (DHT11/DHT22)  
- Distance measurement using ultrasonic sensors  
- Light detection using LDR  
- Object detection using IR sensors  
- Soil moisture monitoring  
- Using potentiometers for analog control  

---

### ⚙️ Actuators & Output Devices
- LEDs and buzzers for basic outputs  
- Servo motors for position control  
- DC motors and speed control  
- Relay modules for switching devices  
- 16×2 LCD display for real-time feedback  
- Matrix keypad for user input  
- Seven-segment display for numeric output  

---

### 🔗 Communication & IoT
- Serial communication (UART)  
- SPI and I2C protocols  
- Bluetooth (HC-05) and BLE basics  
- Wi-Fi connectivity using ESP32  
- MQTT for IoT messaging
