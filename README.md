**Troubleshooting Tips**
 Circuit not working? Double-check wiring and pin connections
 LED/LCD issues? Verify power supply and contrast (for LCD)
 Serial Monitor not showing output? Ensure correct baud rate
 Wi-Fi/MQTT issues? Check network credentials and broker address
 Code errors? Confirm correct libraries are installed
